什么是 Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

本网站即为基于 Hexo 开发：yuanzhibx.github.io

此网站开发环境为 mac , 可能与 windows 系统有所不同，仅供参考。本人新手，如有错误欢迎指正

环境搭建

安装 Hexo 环境

  • Git : https://git-scm.com/
  • Node.js : https://nodejs.org/

安装 Hexo

由于国外镜像源速度过慢，所以使用 npm 安装 cnpm 镜像(淘宝镜像)

npm install -g cnpm --registry="https://registry.npm.taobao.org"

使用 cnpm 安装 Hexo

cnpm install -g hexo-cli

Hexo 博客搭建

创建一个文件夹

mkdir blog

创建一个新的 Hexo 博客

cd your_blog_root/

sudo hexo init

创建一篇新的文章

sudo hexo n "关于我"

文章所在路径

/blog/source/_posts

Hexo 清理缓存

hexo clean

Hexo 生成网页

hexo g

Hexo 本地预览

输入以下命令后可在 http://localhost:4000 中预览网页效果

hexo s

Hexo 部署到 Github

首先需要有一个 Github 的账户并创建一个和你名字相同的仓库，如：
yuanzhibx.github.io

安装 git 部署插件

sudo cnpm install --save hexo-deployer-git

设置 git 地址

cd your_blog_root/

修改 -config.yml 配置

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo: https://github.com/yuanzhibx/yuanzhibx.github.io.git
  branch: master

部署（推送）到 github 远端

sudo hexo d

切换 Hexo 主题

  • 本网站使用主题为 tumo-next，一款很好看的主题

    下载主题

    cd your_blog_root/

git clone https://github.com/TyCoding/tumo-next.git themes/tumo-next

    修改 _config.yml 文件

    theme: tumo-next

mac 远程连接服务器

进入 mac 管理员模式

  1. 在终端输入如下命令
    sudo su -
  2. 随后键入密码既可

连接至服务器

  1. 在终端输入如下命令
    ssh -p 端口 用户名@IP地址
  2. 随后键入密码既可
