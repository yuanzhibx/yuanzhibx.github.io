Linux 常用命令

Posted on 2020-07-03

该文章为本人学习记录使用，整理了日常使用的命令。如有错误欢迎指正。

什么是 Linux

Linux，全称 GNU/Linux，是一套免费使用和自由传播的类 UNIX 操作系统，其内核由 Linus Benedict Torvalds 于1991年第一次释出，它主要受到 Minix 和 Unix 思想的启发。它能运行主要的 Unix 工具软件、应用程序和网络协议。它支持32位和64位硬件。Linux继承了Unix以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统。

常用命令

ls

列出文件或者目录的信息，目录的信息就是其中包含的文件。

cd

更换当前目录。

mkdir

创建目录。

grep

使用正则表示式进行全局查找并打印。

find

文件搜索。可以使用文件的属性和权限进行搜索。

  • 与时间有关的选项

rmdir

删除目录，目录必须为空。

cp

复制文件。如果源文件有两个以上，则目的文件一定要是目录才行。

rm

删除文件。

mv

移动文件。

ps

查看某个时间点的进程信息。
