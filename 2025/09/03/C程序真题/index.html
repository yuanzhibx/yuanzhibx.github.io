🎯 高频考点总结

  • 顺序程序：数据类型字节数与范围、表达式优先级与结合性、printf/scanf 格式化。
  • 循环与控制：for/while/do…while 的执行次序与终止条件，break/continue 的区别，自增自减的先后性。
  • 数组与统计：冒泡/选择/插入排序；求均值并统计大于均值的个数；水仙花数、含“6”的数、素数判断。
  • 字符串：存储长度=字符数+1（含 '\0'）；大小写转换（toupper/tolower）。
  • 函数：值传递陷阱（形参修改不影响实参）、函数调用分支。
  • 文件 I/O：fopen/fclose/fprintf/fscanf，顺序读写与格式化输入输出。
  • 综合建模：鸡兔同笼、换钱组合等枚举问题。

第3章 最简单的 C 程序设计——顺序程序设计

3.2 数据类型与表达式

  • 真题

    • 【2019 简答】常见基本数据类型与内存大小。
    • 【2020 程序】for 循环次数与表达式求值。
    • 【2021 程序】综合表达式与输出。
    • 【2022 选择】合法标识符；运算顺序；字符串占用长度。

  • 答案

    • 标识符：字母/数字/下划线，不能以数字开头，不得与关键字冲突。
    • 运算顺序：a++ 先用后加，++a 先加后用；算术、关系、逻辑有优先级与结合性。
    • 字符串字面量长度 = 可见字符数 + 1（结尾 '\0'）。
