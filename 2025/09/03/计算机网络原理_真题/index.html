🎯 高频考点总结

  • 体系结构：几乎年年考，OSI 七层、TCP/IP 四层、五层协议结构；各层对应功能。
  • 子网划分：高频大题，考查掩码、网络地址、可分子网数、可用主机范围。
  • 数据链路层：链路层功能、CSMA/CD 协议。
  • 交换机/以太网：交换机转发原理、MAC 地址表。
  • 传输层：TCP 与 UDP 的区别，TCP 可靠传输机制。
  • 应用层：DNS 系统、HTTP、邮件协议，尤其 2025 年考了 DNS 缓存作用。

第1章 概述

1.1 计算机网络的作用

  • 真题

    • 【2019 简答】计算机网络在信息时代中的作用。
    • 【2020 简答】计算机网络的主要功能。
    • 【2023 简答】计算机网络的定义与发展阶段。

  • 答案

    • 网络作用：资源共享、数据通信、分布式处理、可靠性提升。
    • 定义：由若干自治计算机通过通信链路互联而成。
    • 发展阶段：单机通信 → 局域网 → 互联网。

1.7 网络体系结构

  • 真题

    • 【2019 选择】五层协议各层功能。
    • 【2021 简答】TCP/IP 四层与 OSI 七层的关系。
    • 【2022 选择】模型层次结构。
    • 【2023 简答】TCP/IP 各层协议。
    • 【2025 简答/综合】五层结构及作用，结合各层协议举例。

  • 答案

    • 五层结构：应用层、运输层、网络层、数据链路层、物理层。
    • TCP/IP 四层：应用、传输、网际、网络接口。
    • 功能：应用（服务）、传输（端到端）、网络（路由）、链路（成帧）、物理（比特）。

  • 答案

第3章 数据链路层

3.1 功能

  • 真题

    • 【2021 简答】数据链路层的功能。
    • 【2022 简答】数据链路层的功能。

  • 答案

    • 功能：成帧、差错检测、可靠传输、流量控制。

3.3 CSMA/CD

  • 真题

    • 【2022 选择】CSMA/CD 协议的特点。

  • 答案

    • 特点：先听后发，边发边听，冲突退避重传。

第4章 网络层

4.2 IP 地址与子网划分

  • 真题

    • 【2020 综合】子网划分。
    • 【2021 综合】子网划分。
    • 【2022 综合】子网划分。
    • 【2023 综合】子网划分。
    • 【2025 综合】子网划分：/16 划分为 16 个子网，写出地址范围与掩码。

  • 答案

    • 掌握子网掩码：CIDR 表示。
    • 子网数 = 2^n，主机数 = 2^m - 2。
    • 例：129.19.0.0/16 → 划分 16 个子网，掩码 /24，每个子网 254 主机。

4.6 路由协议

  • 真题

    • 【2025 简答】路由选择与转发的区别。

  • 答案

    • 路由选择：确定路径。
    • 转发：将分组送往下一跳。

第5章 运输层

5.1 TCP/UDP

  • 真题

    • 【2021 简答】TCP 与 UDP 的区别。
    • 【2023 简答】TCP/IP 各层协议。
    • 【2025 简答】运输层的作用。

  • 答案

    • TCP：面向连接，可靠，字节流。
    • UDP：无连接，不可靠，面向报文。
    • 运输层功能：端到端通信，复用分用，差错控制。

第6章 应用层

6.1 DNS

  • 真题

    • 【2025 简答】DNS 系统中高速缓存的作用。

  • 答案

    • 缓存作用：减少查询延迟，减轻上级服务器压力，提高解析效率。

6.4 万维网与常见应用

  • 真题

    • 【2023 简答】列举五个常见应用。

  • 答案

    • WWW、Email、DNS、FTP、Telnet。

第7章 数据通信与交换机

7.1 交换机原理

  • 真题

    • 【2023 简答】交换机的基本工作原理。

  • 答案

    • 交换机工作在数据链路层，根据 MAC 地址表转发数据帧，实现局域网互联。
