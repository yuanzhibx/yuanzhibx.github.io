    绪论

    一、RS 解译方法分类

    • 解译：获取信息的过程
    • 处理：信息转换的过程，处理是为了更好解译

    1.1 常规目视解译

    • 最普遍、最基本的解译方法。
    • 主要根据影像的灰阶图形阴影等以及各种物体间影像关系来鉴别物体
    • 与解译人员的专业训练及经验水平有很大关系，受像片分辨率力与人眼分辨率力的影响

    1.2 电子光学解译

    • 利用胶片的影像进行解译
    • 信息量大、影像鲜明丰富（如假彩色合成的方法），使我们可以从某些微弱的信息中识别地物
    • 实际上是目视法的延长，最终要得出判读结果，还需由人来完成，仪器设备本身不能得出最终的结果

    1.3 计算机解译

    • 利用记录的磁带解译，容量大
    • 由计算机进行判读识别，效率高
    • 可以直接显示、打印解译结果

    第一章 RS 图像信息增强的光学处理

    3.1 基础概念

    3.1.1 信号与数据

    • 信号：信息的载体，可转换为可识别的形式（如电磁波信号）。
    • 数据：信号的数字化表示（如二进制数），用于计算机存储和传递信息。

    3.1.2 存储单位

    • 位（bit）：计算机最小存储单位，取值为0或1。
    • 字节（Byte）：1字节=8位，是信息存储的基本单位。
    • 块（Block）：作为单元记录的一组相关数据，读入内存后对应“缓冲区”，适用于大数据量影像处理。
    • 文件（File）：数据管理的基本单位，一幅影像对应一个或多个文件；按存储方式分为连续文件、链接文件、索引文件。

    3.2 CCT磁带

    • 定位：存储遥感图像信息的基础设备。
    • 规格：常见宽×长（如12.7mm×2400m），多为9磁道记录（CT常用），每TM数据为7×3字节。
    • 记录内容：
      1. 标注表（ZD）：记录卫星号、成像时间、影像内容、波段数、扫描行数等。
      2. 注记记录（AN）：卫星胶片四边说明，含成像时间、坐标位置。
      3. 图像数据段（ID）：核心内容，记录每个像素各波段的灰度值。
      4. 辅助记录（SAT）：记录卫星高度、传感器仰角、扫描角等参数，用于后续校正。

    3.3 遥感数据格式

    3.3.1 BIP（波段交叉格式）

    • 存储方式：按像素顺序记录，每个像素的所有波段数据连续存储，适用于快速显示单像素多波段信息。

    3.3.2 BIL（波段行交叉格式）

    • 存储方式：按行记录，每行内各像素的同一波段数据连续存储；与BIP格式结构相似，均由多个文件组成，兼顾波段和行处理效率。

    3.4 数字图像参数

    • 定义：由具有特定灰度值的像素组成的矩阵（如MSS影像为2340×3250矩阵）。
    • 关键参数：
      1. 灰度值（DN值）：无单位整数，反映地物辐射水平，范围通常为0-255（Rmax一般为50-55）。
      2. 空间坐标：像素的行、列或纠正后的地理坐标。
      3. 物理意义：灰度值对应地物的光谱特征（反射/辐射特性），但非地物实际属性。

    3.5 图像处理系统

    3.5.1 硬件组成

    • 核心设备：计算机、输入设备（扫描仪、数模仪）、输出设备（磁带机、彩色显示器、打印机）、存储设备（磁盘、磁带）、显示交互设备（显存器）、专用处理设备（阵列处理器）。

    3.5.2 软件组成

    • 系统软件：操作系统（如Windows、Linux）。
    • 应用软件：数据管理软件、输入输出控制软件、基础图像处理软件（功能含数据文件转换、图像特征提取、辐射/几何校正、增强处理、成果输出）。

    第三章 遥感图像的预处理

    4.1 辐射纠正

    4.1.1 误差来源

    • 成像过程中的干扰因素：大气散射、传感器噪声、卫星速度变化、地形影响等，导致影像灰度值偏离地物真实辐射特性。

    4.1.2 纠正内容

    • 消除辐射失真和噪声，将传感器记录的DN值转换为地物真实辐射值（如反射率、表面温度）。

    4.1.3 辐射定标

    • 定义：将DN值转换为绝对辐射值（如辐射亮度、反射率）的过程。
    • 常用方法：内部定标（利用传感器内部标准源）、外部定标（利用地面标准目标、辐射场），公式参考(L = G \times DN + B)（G为增益，B为偏移量）。

    4.2 几何纠正

    4.2.1 误差来源

    • 传感器姿态（如扫描速度不稳定）、卫星轨道（高度/速度变化）、地球自转/曲率/地形，导致影像几何变形（拉伸、偏移等）。

    4.2.2 处理方法

    • 粗纠正：消除传感器自身误差，由地面站完成。
    • 精纠正：
      1. 控制点校正：选取地面控制点（GCP，需有明显标识、稳定、分布均匀），建立原图像与标准地图的坐标关系，通过多项式模型校正。
      2. 重采样：采用最邻近法、双线性内插法、三次卷积内插法，重新计算纠正后偏移像素的灰度值。

    4.2.3 操作步骤

    • 建立坐标对应关系→选取GCP→精度检查（如RMSE）→数学模型校正→重采样→重复至精度达标。

    4.3 大气纠正

    4.3.1 目的

    • 消除大气散射、吸收对影像灰度的影响，建立DN值与地物真实辐射值的关联。

    4.3.2 常用方法

    1. 辐射传递方程法：模拟电磁波在大气中的传递过程，构建方程求解；精度高但计算复杂。
    2. 野外波谱测试法：同步测量传感器影像与地面地物波谱，通过回归分析消除大气影响；实操性强但受场地限制。
    3. 波段间对比分析法：利用大气影响小的波段（如TM7）校正影响大的波段；简单常用（如TM影像大气校正）。

    4.4 图像镶嵌与裁剪

    4.4.1 图像镶嵌

    • 目的：将多幅影像拼接为大幅影像。
    • 关键步骤：先解决几何配准（同名点匹配），再进行灰度匹配（如直方图匹配、线性拉伸）。

    4.4.2 图像裁剪

    • 目的：去除无用区域，保留目标区域。
    • 操作方法：将边界数据转换为图像坐标，通过掩膜计算（目标区域设1、非目标设0）提取目标影像。

    4.5 图像统计特征

    4.5.1 描述指标

    1. 均值（μ）：影像灰度平均值，反映整体亮度水平，公式(\mu = \frac{1}{N} \sum_{i=1}^N DN_i)（N为像素数）。
    2. 中值（Med）：排序后中间位置的灰度值，受极端值影响小。
    3. 众数（Mode）：出现频率最高的灰度值，可反映优势地物灰度特征。
    4. 标准差（σ）：反映灰度离散程度，σ越大，影像信息差异越大，公式(\sigma = \sqrt{\frac{1}{N} \sum_{i=1}^N (DN_i - \mu)^2})。

    4.5.2 直方图

    • 类型：
      1. 频率直方图：纵轴为灰度级出现频率，对应概率密度曲线。
      2. 累积直方图：纵轴为小于等于某灰度级的频率，对应累积概率密度曲线。
    • 特点：反映灰度分布，理想状态为正态分布；直方图偏移会导致影像偏亮/偏暗，可通过均衡化调整。

    第四章 图像增强处理

    5.1 灰度增强

    5.1.1 线性拉伸

    • 原理：将原灰度范围（a-b）按线性关系拉伸至目标范围（c-d）。
    • 公式：(x_1 = \frac{(d - c)}{(b - a)}(x - a) + c)，适用于整体灰度分布均匀的影像。

    5.1.2 指数拉伸

    • 目的：增强高灰度区域细节。
    • 公式：(x_1 = \frac{d - c}{e^b - e^a}(e^x - e^a) + c)。

    5.1.3 对数拉伸

    • 目的：增强低灰度区域细节，压缩高灰度区域。
    • 公式：(x_1 = \frac{d - c}{\log b - \log a}(\log x - \log a) + c)，适用于暗区信息模糊的影像。

    5.2 比值与差值增强

    5.2.1 比值增强

    • 原理：计算两波段对应像素灰度值的比值（如(R = \frac{DN_i}{DN_j})）。
    • 优势：消除光照条件影响，突出地物差异；可监测地物变化（如植被生长）、识别特定地物（水体、植被）；分为简单比值、组合比值。

    5.2.2 差值增强

    • 原理：计算两波段/时相影像的灰度差，公式(D = a(DN_1 - DN_2) + b)（a、b为调节系数）。
    • 目的：提取变化信息（如土地利用变化）。

    5.3 彩色增强

    5.3.1 真彩色合成

    • 特点：合成影像颜色与地物真实颜色一致/接近，需选择可见光波段（红、绿、蓝）组合。

    5.3.2 假彩色合成

    • 特点：通过非可见光波段（如红外+红+绿）组合形成彩色影像，突出特定地物（如植被呈红色）；常用彩色合成仪实现波段精准叠加。

    5.3.3 密度分割彩色显示

    • 操作：将单波段灰度影像按密度分级，用不同颜色标注各级，通过彩色显示器呈现，便于快速识别地物。

    5.4 平滑处理

    5.4.1 滑动平均法

    • 原理：以窗口内像素灰度平均值作为中心像素值，降低噪声。
    • 局限性：窗口越大，平滑效果越强，但易导致影像模糊。

    5.4.2 中值滤波法

    • 原理：取窗口内像素灰度的中值作为中心像素值。
    • 优势：有效抑制椒盐噪声，同时保留边缘细节。

    5.4.3 局部平均法

    • 特点：带门限值的滑动平均，计算窗口均值后，对比中心像素与均值的差值，超阈值则用均值替换，减少影像模糊。

    5.5 边缘增强

    5.5.1 高通滤波法

    • 原理：抑制低频（平滑区域）、保留高频（边缘），公式(H(u, v) = \frac{1}{1 + (\frac{D_0}{D(u, v)})^2})（(D(u, v))为频率域距离，(D_0)为截止频率）。
    • 目的：突出地物边界。

    5.5.2 方向模板卷积法

    • 原理：用方向模板（水平、垂直等）对影像进行卷积运算，模板元素按灰度变化规律取值，在目标方向上卷积值最大。
    • 目的：增强特定方向的边缘。

    5.6 图像变换

    5.6.1 K-L变换（主成分变换）

    • 原理：线性正交变换，通过协方差矩阵的特征值和特征向量构建变换矩阵，公式(Y = A X)（A为特征向量矩阵，X为原影像向量，Y为变换后影像）。
    • 特点：变换后各主成分互不相关，第一主成分集中大部分信息，减少数据量；用于多波段影像去冗余、信息提取。

    5.6.2 RGB to IHS变换

    • 原理：将RGB彩色空间（红、绿、蓝）转换为IHS空间（亮度I、色调H、饱和度S）。
    • 优势：便于单独调整亮度或色调，公式参考(I = \frac{R + G + B}{3})。

    5.6.3 其他变换

    • 缨帽变换：突出植被、土壤信息；小波分析：多尺度分解影像，兼顾细节与整体。

    第五章 RS 图像的分类处理

    6.1 分类原理

    • 核心逻辑：通过地物光谱特征，在图像特征空间中划分互不重叠的类别区域，将像素分配至对应类别，实现地物信息提取（如面积统计、类型划分）。

    6.2 分类方法

    6.2.1 监督分类

    • 原理：“先学习后分类”，选取有代表性的训练区（地物单一、已知类别、分布均匀），训练分类器并建立判别规则，再对全图分类。
    • 操作步骤：打开影像→选择训练区→训练分类器→分类→精度评价→结果输出。
    • 判别方法：
      1. 最大似然法：计算像素属于各类别的概率，分配至概率最大的类别。
      2. 最小距离法：计算像素与各类别中心的距离，分配至距离最近的类别。
      3. 特征曲线法：对比像素光谱曲线与类别特征曲线，相似度最高的为目标类别。
    • 优缺点：
      • 优点：可自定义类别，避免无用类别；可验证训练样本，减少错分。
      • 缺点：主观因素强；训练区选取耗时；难以覆盖稀有类别。

    6.2.2 非监督分类

    • 原理：“先分类后学习”，根据影像像素的统计特征（如灰度分布）自动聚类，无需训练区，适用于未知地物分布的场景。
    • 常用方法：
      1. 聚类分析法：计算像素间距离（如欧氏距离），合并距离最小的类别，重复至达到预设类别数。
      2. ISODATA法：迭代自组织数据分析，动态调整类别中心，拆分/合并类别（如最大类别直径超阈值则拆分），优化分类结果。
    • 优缺点：
      • 优点：无需地物先验知识，减少人工干预；可发现未知地物类别。
      • 缺点：类别需后续人工识别；受光谱变异（时间、地形影响）干扰大；难以区分光谱相似地物。

    6.3 精度评价

    6.3.1 误差矩阵（混淆矩阵）

    • 定义：n×n矩阵（n为类别数），行代表实际类别，列代表分类类别，元素(P_{ij})为实际i类被分为j类的像素数，反映分类错分/漏分情况。

    6.3.2 核心评价指标

    1. 总体精度（OA）：正确分类像素数占总像素数的比例，公式(OA = \frac{\sum_{i=1}^n P_{ii}}{\sum_{i=1}^n \sum_{j=1}^n P_{ij}})。
    2. 用户精度（UA）：分类为i类的像素中实际为i类的比例，公式(UA_i = \frac{P_{ii}}{\sum_{j=1}^n P_{ji}})，反映分类结果可靠性。
    3. 生产者精度（PA）：实际为i类的像素中被正确分类的比例，公式(PA_i = \frac{P_{ii}}{\sum_{j=1}^n P_{ij}})，反映地物识别率。
    4. Kappa系数：考虑偶然一致的精度指标，K≥0.8为合格，公式：
      (K = \frac{N \sum_{i=1}^n P_{ii} - \sum_{i=1}^n (\sum_{j=1}^n P_{ij} \sum_{j=1}^n P_{ji})}{N^2 - \sum_{i=1}^n (\sum_{j=1}^n P_{ij} \sum_{j=1}^n P_{ji})})（N为总像素数）。

    6.3.3 采样方法

    • 简单随机采样：适用于类别分布均匀的场景。
    • 系统采样：按固定间隔采样，保证采样均匀性。
    • 分层采样：按地物类型/行政区分层后采样，提高针对性。
    • 集群采样：选取连续区域作为样本，适用于大面积同类地物。

    七、核心知识点总结

    1. 处理流程：遥感图像从解译到应用，需经历“解译→预处理（辐射/几何/大气纠正）→增强处理→分类→精度评价”的完整流程。
    2. 关键技术：数字图像处理是核心，涵盖灰度增强、图像变换、监督/非监督分类等；光学处理是传统基础手段，适用于简单影像调整。
    3. 核心指标：DN值（灰度值）、Kappa系数（分类精度）、标准差（影像信息差异）是评价图像质量和处理效果的关键指标。
    4. 应用逻辑：所有处理手段均以“突出地物信息、提高识别精度”为目标，需根据影像类型（多波段/单波段）和应用需求（地物监测/面积统计）选择适配方法。
