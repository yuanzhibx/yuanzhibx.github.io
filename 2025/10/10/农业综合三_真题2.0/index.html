【2019 - 2025】真题 2.0

2019

一、选择题

（一）数据库部分（10*1）

  1. DBS 包括 DB 和 DBMA
  2. 外模式、内模式、模式
  3. 关系代数的几个符号
  4. 数据库特征

（二）计算机网络部分（10*1）

  1. 最早出现的网络是？
  2. 子网数与主机数的算法？
  3. 协议的描述
  4. 协议的三要素

二、简答题

（一）C语言部分

  1. C 语言中常用的数据类型、符号表示、所占内存。
  2. for 语句的表达式表示以及实现顺序。

（二）数据库部分

  1. 数据库管理系统的功能；
  2. 简述基本表的定义以及视图定义，二者的区别与联系；
  3. 关系代数的基本操作有哪些？

（三）计算机网络部分

  1. 简述计算机网络的概念。
  2. TCP、UDP 的相同点与不同点有哪些？
  3. 网络的各层结构名称，数据链路层功能有哪些？

三、输出程序结果（4*4）

四、程序设计题（3*8）

  1. 输入华氏温度，输出摄氏温度。
  2. 输入 6 个数，按从小到大的顺序输出。

五、综合题

（一）计算机网络部分

  1. 填写一个路由表（子网掩码、MAC 地址之类的）。

（二）数据库部分

根据要求写 SQL 语句（10）

  1. 写一个查询语句
  2. 写一个更新语句
  3. 写一个授权语句
  4. 写一个建立视图语句

2020

一、选择题（数据库、计算机网络各十道题，30*1）

  1. 因特网最早起源；
  2. IPV6 位数 64 位；
  3. 哪个为非通讯工具；
  4. SMTP/POP 协议；
  5. RIP 协议设置的最大距离为 15；
  6. E-R 图属于概念模型；
  7. 模式内模式外模式；
  8. DBS包括DB、DBMS；

二、简答题（7*8）

  1. 什么是主码、外码、候选码？
  2. 数据库管理系统的内容包括？功能是什么？
  3. 关系数据库的功能是什么？内模式、模式、外模式定义是什么？
  4. 简述以太网交换机是怎样转发帧的？转发表是怎样建立的？
  5. 简述 TCP、UDP 的相同点和不同点，简述路由器组成及功能。
  6. 简述计算机网络的分类。
  7. 硬件地址与 IP 地址有什么异同？

三、输出程序结果题（4*4）

  1. 加减乘除后变量的值。
  2. for 循环判断几次停住。

四、C语言程序设计题（3*8）

  1. 输出 100 以内含 6 的数；
  2. 输出 100-1000 内的水仙花数；
  3. 输入 10 个数，按从小到大的顺序输出；

五、综合题（3*8）

（一）数据库部分

  1. 解释 SQL 语句的含义：增加查询更新删除
  2. 写 SQL 语句：增加查询更新删除

（二）计算机网络部分

  1. 将普通地址（C 类）的子网 192.168.118.0 划分为 4 个子网。

2021

一、选择题（C 语言、数据库、计算机网络各十道题，30*1）

（一）C语言部分

  1. C 语言变量命名规则
  2. 从 mian 函数开始运行
  3. 运算顺序

（二）数据库部分

  1. 数据库设计的三个阶段
  2. DBS、DBMS、DB 的包含关系
  3. 数据字典是什么
  4. 数据库存储的是数据及其联系

（三）计算机网络部分

  1. 因特网实际是（分组交换网络）
  2. HTML 是什么意思
  3. 从上到下第一个进行端到端通信的是哪一层
  4. Ipv4 的位数 32 位
  5. Ipv6 的字节数是 16B

二、简答题（7*8）

  1. DBMS 是什么？有什么作用？
  2. 关系的约束包括哪三类？实体约束是什么？
  3. 基础表和视图有什么联系和区别？
  4. 简述计算机网络按分布范围的分类，5G 属于什么网络？举两个局域网的例子。
  5. CSMA/CD 的名称是什么？适用于什么网络？
  6. 简述一个路由器收到一个分组后，转发该分组的过程。
  7. 当你使用计算机浏览新浪网时，在浏览器输入 www.sina.com，到你的浏览器完整显示出这个网页的过程中，TCP/IP 模型的应用层到网络层分别应用了什么协议？

三、输出程序结果（4*4）

  1. 变量运算输出

  2. for 循环累加

  3. 函数调用与值传递

四、程序设计题（3*8）

  1. 输入五个数，请按从大的到小的顺序输出。
  2. 输入一元二次方程的系数，输出该方程的解。
  3. 输入一个数，判断这个数是不是素数。

五、综合题（3*8）

（一）数据库部分

  1. 解释 SQL 语句的含义：
    • SQL 语句定义了一个视图，解释含义；
    • SQL 语句修改工人的部门。
  2. 写 SQL 语句：
    • 将 STUDENT 的所有功能授权给“ZHAO”；
    • 将“张三，18，男，计算机系，20210913”插入 STUDENT 表。

（二）计算机网络部分

有五个 IP 地址，A、127.0.0.0，B、128.256.12.15，C、193.168.1.255，D、225.162.123.12，E、125.26.0.12，请分析这五个 IP 地址能否分配给主机？如不能，请说明理由。

2023

一、选择题（30个选择题，C语言、数据库、计算机网络各有10个，30分）

二、简答题（56分）

  1. 简述数据库定义，数据库的基本特征
  2. 简述数据模型的定义，数据模型的功能和数据模型的三要素
  3. 为什么数据库设计先要进行需求分析
  4. 计算机网络定义，计算机网络的发展阶段
  5. tcp/ip模型分为哪几层，每层分别对应那几个协议
  6. 交换机的基本工作原理
  7. 网络层和传输层分别有什么作用

三、输出程序结果（4*4）

  1. 条件判断修改变量值
  2. 函数调用计算平方和
  3. do-while 循环输出自减值

四、程序设计题（3*8）

  1. 输入一个大于零的数，判断输出这个数的位数
  2. 对于一个给定的整形数组（长度为10）求其平均数，并将该数组中大于平均数的个数输出
  3. 有一张100元的人民币，把它换成100张五元，一元，0.5元数值的钱且每种数值的钱至少有一张，有几种组合，分别输出出来

五、综合题（24分）

（一）数据库部分

  1. 看sql语句写意思（共四小问，回忆起两个）

  2. 基于顾客表（顾客号 顾客名 年龄 性别 ）、商品表（商品号 商品名 商品订单）、订购表（顾客号 商品号 数量）：

    • 创建一个视图gm_view，查询包括顾客号 顾客名 订购的商品号 数量；
    • 选择女性用户的顾客号 商品名 商品号 数量

（二）计算机网络部分

对c类网络192.168.118.0 创建四个子网，给出有效的网络地址范围，对应的网络掩码

2025

一、选择题

（一）数据库部分

  1. 主要考第二章和第三章基本知识点包括数据库功能，数据库关系运算、查询、操纵语言(比较简单)

（二）计算机网络部分

  1. 考计算机网络结构及对应作用(五层体系结构和对应作用)

（三）C语言部分

  1. 都是基本知识点，基本都会碰到过

二、简答题

（一）计算机网络部分

  1. 语义，语法，同步的定义。
  2. DNS系统中高速缓存的作用

（二）数据库部分

  1. 写语句，比如查询操作，插入删除修改，查询要会用order by和group by having 语句

三、输出程序结果（4*4）

  1. 一般是输入输出函数
  2. 输入一串大小写混合的英语语句，结果是输出之后全部变成大写

四、程序设计题（3*8）

  1. 常规题
  2. 写一个一元二次方程式，题目给x让你输出y
