第一章 绪论

第一节 遥感与遥感技术

一、遥感概述

遥感技术是20世纪60年代兴起并迅速发展起来的一门综合性探测技术，它是在航空摄影测量的基础上，随着空间技术、电子计算机技术等现代科技的发展以及地学、生物学、环境科学等学科的需要发展形成的。其快速发展的重要原因与地球资源卫星（ERTS-1）试验成功有关。通过卫星，可以对地球表面任何一个角落反复探测，获取资源与环境的海量信息，使遥感研究成为世界性课题。

（一）遥感的概念“遥感（Remote Sensing）”一词最早由美国地理学家伊瑞林·普鲁特等于1960年提出。

  • 含义：遥感【25年真题：名词解释】 即遥远的感知，通常指空对地遥感，即不直接接触物体本身，通过传感器探测和接收来自目标物体的电磁波、电场、磁场等信息，经传输和处理分析，识别物体属性和分布特征。
  • 遥感技术：是指利用不同高度平台上的各种传感器接收地物电磁波信息，并进行加工处理，从而远距离探测和识别地物及其特性的综合技术。

（二）遥感技术的特点

  1. 宏观性和综合性强

    • 强调大范围、大尺度获取自然资源与生态环境信息。
    • 航空相片可提供大比例尺景观实况，卫星图像覆盖范围更大，适于区域、全球性研究。
    • 可避免障碍阻拦，影像清晰完整。
    • 涉及电磁学、仪器学、信息学、图像识别等多学科。

  2. 综合效益高

    • 工作进度快、精度高、费用省。
    • 能调查难以到达区域，发现肉眼难察觉的细微变化。
    • 在社会、经济、生态环境方面均具高效益。

  3. 信息量大，技术先进，准确性高

    • 可获取紫外、可见光、红外、微波等多波段信息。
    • 不仅能摄影，还能扫描，信息量远超传统方法。
    • 图像细节清晰，能准确反映地貌、植被、水文、道路等信息。

  4. 获取信息快，更新周期短，动态信息丰富

    • 遥感成像为瞬时成像，能获取同一时刻大面积景观。
    • 卫星重复成像周期短，如陆地卫星4/5号每16天覆盖全球，气象卫星每天覆盖全球。
    • 可监测病虫害、洪水、污染、火山、地震等，为灾害预警提供依据。

  5. 应用领域广泛

    • 广泛应用于农业、林业、地质、水文、军事等领域。

第二节 遥感技术系统组成及其遥感过程

一、遥感技术系统组成

遥感技术系统主要由遥感平台、传感器和地面控制系统组成。

（一）遥感平台

  • 近地面平台：汽车、轮船、高塔、支架等。既独立使用，也作为航空、航天遥感的校准。
  • 航空平台：飞机、气球等。分辨率高，机动灵活，但成本较高。
  • 航天平台：火箭、人造卫星、航天飞机等。优点是宏观性、动态性、快速性和经济性。

（二）传感器

  • 按工作方式分：主动式（自带辐射源，如雷达、激光雷达）、被动式（接收自然辐射或地物自辐射，如相机、扫描仪）。
  • 按记录方式分：成像方式（照片、影像）、非成像方式（辐射计、光谱仪）。
  • 成像方式又分为：摄影方式、扫描方式。

（三）地面控制系统

  • 指挥与控制传感器和平台，接收数据并进行预处理。
  • 现代遥感一般由计算机系统完成姿态控制、数据接收、预处理和分发。

二、遥感过程

遥感过程是指遥感信息的获取、传输、处理以及分析判读和应用的全过程。其内容包括：

  • 遥感信息源（或地物）的物理性质、分布及运动状态；
  • 环境背景及电磁波光谱特性
  • 大气的干扰和大气窗口
  • 传感器的分辨能力、性能和信噪比
  • 图像处理及识别
  • 人们的视觉生理和心理及其专业素养

按照技术实施划分，遥感技术系统主要由以下四部分组成：

  1. 遥感试验：提供地物光谱特性与各类校正所需信息，是整个遥感过程的基础。
  2. 遥感信息获取：以平台与传感器为物质保证，是系统的中心工作。
  3. 遥感信息处理：用于消除干扰、进行校正与增强、提取信息，是保证质量与精度的重要环节。
  4. 遥感信息应用：遥感的最终目的，按专业目标选择适宜信息与方法，以取得社会与经济效益。

第三节 遥感的发展概况及展望

一、遥感的发展概况

  1. 萌芽及初期阶段

    • 人类通过感官认识环境，后有伽利略望远镜（1610）、气球摄影（1858）。

  2. 航空遥感（飞机遥感）阶段

    • 1909年首次飞机航摄，1913年进入应用；一战中快速发展，战后用于测绘、地质等。
    • 二战中发展了红外与雷达技术。

  3. 航天遥感（卫星遥感）阶段

    • 二战后探空火箭拍摄地表。1957年苏联发射第一颗人造卫星，1959年实现月球探测，1960年美国发射气象卫星。
    • 1972年美国发射陆地卫星，1986年法国发射SPOT卫星。

  4. 我国遥感发展概况

    • 1970年我国成功发射第一颗人造卫星，标志着中国进入航天遥感时代。
    • 1970年代末至1980年代，开展了国土资源普查、农业遥感试验、航空摄影测量等，初步形成应用体系。
    • 1980年代以来，相继研制并应用了多光谱扫描仪、成像光谱仪、合成孔径雷达（SAR）等设备。
    • 进入21世纪后，风云系列气象卫星、资源系列卫星、环境减灾系列卫星、高分系列卫星相继发射，实现了多领域、多任务的遥感应用，缩小了与国际先进水平的差距。

二、发展趋势与展望

  1. 新一代传感器的研制，以获得分辨率更高、质量更好的遥感图像和数据
    • 空间、光谱、时间分辨率不断提高，发展成像光谱仪、SAR、激光雷达。
  2. 遥感信息处理技术的发展趋势
    • 引入小波、分形、神经网络等方法，多源数据融合，推动3S技术一体化。
  3. 遥感应用领域拓展
    • 由单一信息源 → 多源复合，由静态 → 动态，由定性 → 定量，由局部 → 全球。
  4. 地理信息系统（GIS）和土地管理信息系统（LIS）的发展与应用是遥感发展的新进展和新动向

第四节 遥感技术的应用

一、在资源调查中的应用

  1. 土地和土壤资源调查
    • 土地资源、土地利用现状、土壤资源、土壤退化（干旱、盐化、沙化、侵蚀）等的调查与监测；典型做法为利用多景卫星影像进行类型划分、面积量算与制图
  2. 地质矿产资源调查
    • 用于基础地质、矿产地质、工程/地震/灾害地质等综合调查；凭借宽视域与客观成像识别区域构造、断裂展布与走向等，为跨区域乃至全球的地质研究提供依据
  3. 水文与水资源调查
    • 开展水资源调查、流域规划、水土流失、冰雪监测以及海岸带与浅海地形、海洋调查等；在高寒与荒漠区可有效补充地面考察，修正/新增湖泊信息并判别淡水/咸水

二、在环境监测评价及对抗自然灾害方面的应用

  1. 环境监测评价
    • 对生态环境变化、大气/水体/土地/海洋污染进行快速、客观、形象的监测与判读，定位污染源、判别污染物性质并跟踪其时空变化，为评价与治理提供依据（如热红外扫描判读污染异常点、绘制污染判读图）
  2. 对抗自然灾害
    • 依托气象卫星与多时相遥感开展灾害性天气监测、洪水预警、滑坡/泥石流调查、地震与地壳形变监测、灾后损失评估与重建规划等；“风云”系列等业务化观测显著提升时效性与准确性

三、在区域分析及建设规划方面的应用

  1. 区域综合调查与规划
    • 以遥感图像的真实、客观、形象为基础，开展区域资源与环境综合调查，支撑分层次的近期—中期—远期规划与产业结构优化（如沿海滩涂资源调查与动态监测系统建设）。
  2. 城市遥感与城建管理：
    • 提供城市土地利用现状、用地分析、环境监测与评价、道路交通与人口、城市生态等基础信息；与城市 GIS集成，用于总体规划与辅助决策（如“北京航空遥感（8301 工程）”形成多项研究成果并推广至各大中城市）。

四、在全球性宏观研究中的应用

  1. 全球变化与圈层系统研究
    • 面向岩石圈—大气圈—水圈—生物圈的宏观、整体性研究；利用遥感开展板块运移与深大断裂活动、环形构造成因等地学问题研究。
  2. 全球气象与灾情预报 / 海洋与冰冻圈监测
    • 基于气象卫星资料进行全球气象研究与灾情预报；开展海洋动力学与地表固态水（冰川）变化监测及全球环境监测。
  3. 国际协同
    • 参与 ISY、IGBP、国际减灾十年 等合作与项目，推进全球尺度的遥感应用与数据产品建设。

五、在农业、林业、畜牧业上的应用

  1. 资源调查
    • 土地资源/利用现状、土壤类型与低产田、林地与森林蓄积量、草场资源与水资源等调查与综合评价，形成基础图件与数据。
  2. 资源监测（动态监测）
    • 作物长势、土地沙化/盐渍化、渔场与鱼群、农业用地污染、草地退化、森林长势等持续性、动态性监测，为生产管理与决策提供时序信息。
  3. 生物量估测（估产）
    • 结合农学与环境因子，估测小麦、水稻、玉米、棉花等作物产量以及森林蓄积量、草场产草量等；多国与国内试验表明总体精度**可达或超过 90%**。
  4. 灾害预报与评估
    • 监测病虫害、草场雪灾、森林火灾、洪水等，进行预警、受灾面积测定与灾后评估；优势在于大范围、及时性强，利于制定防治与应急措施。
  5. 特殊应用
    • 如农业人口与农村交通网调查、水土流失监测、野生动物调查等。

第二章 遥感的物理基础

第一节 电磁波与电磁波谱

一、电磁波及其特性

（一）电磁波
根据电磁场理论，电与磁是紧密联系的两种运动形式。

变化的电场能在其周围激起磁场的变化，变化的磁场也能在其周围激起电场的变化；这种交变电磁场由近及远在空间传播的过程称为电磁波。γ 射线、X 射线、紫外线、可见光、红外线、微波、无线电波等本质相同，仅因频率（或波长）不同而性质各异。

（二）电磁波的特性

  1. 电磁波是一个横波
  2. 电磁波的波动性
  3. 电磁波的粒子性（波粒二象性）
  4. 电磁波的叠加原理
  5. 电磁波的相干性和非相干性
  6. 电磁波的衍射与偏振
  7. 电磁波的多普勒效应

二、电磁波谱

（一）定义
将各类电磁波按波长（或频率）依次排列形成电磁波谱。

（二）共同性

  1. 各类电磁波在真空（或空气）中传播速度相同，都等于光速c（3*10¹⁰ cm/s）
  2. 遵守同一的反射、折射、干涉、衍射、偏振定律。

（四）各谱段要点

  1. 紫外（0.01–0.38 μm）：**<0.3 μm** 基本被大气吸收；0.3–0.38 μm 到达地面能量少，常用于碳酸盐岩分布、海面油膜等监测。
  2. 可见光（0.38–0.76 μm）：摄影/扫描常用，颜色信息丰富，是地物识别的重要波段。
  3. 红外（0.76–1000 μm）：
    • 近红外（0.76–3 μm）：又称反射红外，常用于摄影与扫描（感材摄影约 0.76–1.3 μm）。
    • 中/远红外：遥感多用3–5 μm与8–14 μm，可昼夜成像，对温度敏感。
  4. 微波（1 mm–1 m）：可主动/被动成像，全天候、全天时，对植被/冰雪/土壤等表层有一定穿透能力。

（五）单位与换算
无线电波常用 km、m；微波用 cm、mm；红外用 μm；可见光/紫外用 nm 或 μm。
换算：

  • 1 nm = 10⁻³ μm = 10⁻⁷ cm = 10⁻⁹ m；
  • 1 μm = 10⁻³ mm = 10⁻⁴ cm = 10⁻⁶ m。

三、电磁辐射源

（一）自然辐射源

自然辐射源主要包括太阳辐射和地物的热辐射。

  • 太阳辐射是可见光及近红外遥感的主要辐射源。
  • 地球是远红外遥感的主要辐射源。
  1. 太阳辐射：一般称为短波辐射。
    • 太阳表面温度约 6000 K，大气上界的太阳光谱辐照度近似于黑体 5900 K 的分布，能量主要集中在 0.3–3 μm；大气上界对正对太阳的单位面积接收的辐射功率（太阳常数）约 1385±7 W·m⁻²。
  2. 地球的电磁辐射：地球辐射可分为两部分，短波（0.3~2.5 μm）和长波（6 μm以上）
    • 地表平均温度约 300 K，辐射峰值波长近 9.66 μm（约 9–10 μm）。3 μm 以下传感器接收为地物对太阳辐射的反射为主；6 μm 以上以地物自身热辐射为主；3–6 μm 两者均需考虑。

第二节 地物的光谱特性

自然界中任何地物都具有其自身的电磁辐射规律，如具有反射，吸收外来的紫外线、可见光、红外线和微波的某些波段的特性；它们又都具有发射某些红外线、微波的特性；少数地物还具有透射电磁波的特性，这种特性称为地物的光谱特性。

一、地物的反射光谱特性

当电磁辐射能量入射到地物表面上将会出现三种过程：

  • 一部分入射能量被地物反射；
  • 一部分入射能量被地物吸收，成为地物本身的内能或部分再发射出来；
  • 一部分入射能量被地物透射。

  • Pᵣ：地物的透射能量。

（ 一 ） 地物的反射率

  • 反射率（或反射系数或亮度系数）来表示地物对某一波段电磁波的反射能量与入射的总能量之比。
  • 在遥感图像上色调的差异是判读遥感图像的重要标志。

（ 二 ） 地物的反射光谱

  • 地物的反射率随入射波长变化的规律叫做地物反射光谱。
  1. 雪：
    • 雪的反射光谱和太阳光谱很相似，在 0.4~0. 6µm 波段有一个很强的反射峰，反射率几乎接近100%,因而看上去是白色。
  2. 沙漠：
    • 在橙光波段0. 6µm附近有一个强反射峰，因而呈现出橙黄色，在波长达到0. 8µm以上的波长范围，其反射率比雪还强
  3. 湿地：
    • 潮湿地在整个波长范围内的反射率均较低，当含水量增加时，其反射率就会下降，尤其在水的各个吸收带处，反射率下降更为明显。因而，在黑白相片上，其色调常呈深暗色调。
  4. 湿地：
    • 在蓝光波段（中心波长为0. 45µm）和红光波段（中心波长为0. 65µm）上有两个吸收带，其反射率较低，在两个吸收带之间，即在0. 55µm附近有一个反射峰，这个反射峰的位置正好处千可见光的绿光波段，故而叶子的天然色调呈现绿色。大约在O. 7µm附近，由于绿色叶子很少吸收该波段的辐射能，其反射率骤然上升，至l.lµm近红外波段范围内反射率达到高峰。
  • 同类地物的反射光谱是相似的，但随着该地物的内在差异而有所变化。这种差异可用来识别不同的植物类型。
  • 研究地物的光谱特性，还应考虑其时间特性和空间特性的变化。

二、地物的发射光谱特性

任何地物当温度高于热力学温度0K时，组成物质的原子、分子等微粒在不停地做热运动，都有向周围空间辐射红外线和微波的能力。

通常地物发射电磁辐射的能力是以发射率作为衡量标准。地物的发射率是以黑体辐射作为基准。

（ 一 ） 黑体辐射【25年真题：名词解释 / 简答】

  • 黑体：
    1. 黑体是一个理想的辐射体，黑体也是一个可以与任何地物进行比较的最佳辐射体。
    2. 是“绝对黑体”的简称，指在任何温度下，对千各种波长的电磁辐射的吸收系数恒等于1 (100%)的物体。
  • 黑体辐射：黑体的热辐射称为黑体辐射

（ 二 ） 黑体辐射定律

普朗克定律：表示出了黑体辐射通量密度与温度关系以及按波长分布的情况

  • 黑体辐射的三个特性
    1. 辐射通量密度随波长连续变化，每条曲线只有一个最大值。
    2. 温度愈高，辐射通量密度也愈大，不同温度的曲线是不相交的
    3. 随着温度的升高，辐射最大值所对应的波长移向短波方向

斯特藩–玻尔兹曼定律：黑体总辐射通量密度随温度的增加而迅速增大，它与温度的四次方成正比。

维恩位移定律：随着温度的升高，辐射最大值对应的峰值波长向短波方向移动。

（ 三 ） 地物的发射率和基尔霍夫定律

  • 发射率 ε：指地物的辐射出射度与同温度的黑体的辐射出射度的比值

  • 地物的发射率随波长变化的规律称为地物的发射光谱。

三、地物的透射光谱特性

  • 有些地物（如水体和冰）具有透射一定波长的电磁波能力，通常把这些地物叫做透明地物。地物的透射能力一般用透射率表示。
  • 透射率就是入射光透射过地物的能量与入射总能量的百分比

四、地物光谱特性与遥感图像

在遥感图像上，目标地物特征包括 “色、形、位” 三大类，它们决定的光谱特征与遥感图像之间具有紧密的联系。

  1. 色

    • 指目标地物在遥感图像上呈现的颜色特征，包括目标地物的颜色、阴影、纹理等。

  2. 形

    • 指目标地物在遥感图像上表现的形状特征，包括目标地物的形状、大小等。

  3. 位

    • 指目标地物在遥感图像上的空间位置特征，包括目标地物的分布位置、排列方式等。

第三节 大气和环境对遥感的影响

一、大气成分与结构

（ 一 ） 大气成分：

地球大气是由多种气体、固态及液态悬浮的微粒混合组成的。

  • N₂、O₂、H₂O、CO、CO₂、N₂O、CH₄、O₃

（ 二 ） 大气结构

  1. 对流层：现代航空遥感主要活动的区域
  2. 平流层
  3. 中间层
  4. 热层（增温层）：人造地球卫星绕地球运行的主要空间
  5. 大气外层

二、大气对太阳辐射的影响

（ 一 ） 大气的吸收作用

  1. 氧（O₂）：大气中氧含量约占21%,它主要吸收小千0. 2µm的太阳辐射能量，在波长0. 155µm处吸收最强
  2. 臭氧（O₃）：大气中臭氧的含量只占o. 01%0.1%,但对太阳辐射能量吸收很强。臭氧有两个吸收带：一个波长0.20. 36µm的强吸收带；另一个波长为0. 6µm附近的吸收带，该吸收带处千太阳辐射的最强部分，因此该带吸收最强。
  3. 水（H₂O）：水在大气中以气态和液态的形式存在，它是吸收太阳辐射能量最强的介质。从可见光、红外直至微波波段，到处都有水的吸收带，主要吸收带是处于红外和可见光中的红光波段，其中红外部分吸收最强。
  4. 二氧化碳（CO₂）：大气中二氧化碳含量占0. 03%,它的吸收作用主要在红外区内。由于太阳辐射在红外区能量很少，因此对太阳辐射而言，这一吸收带可忽略不计。
  5. 尘埃：对太阳辐射也有一定的吸收作用，但吸收量很少，当有沙暴、烟雾和火山爆发等发生时，大气中尘埃急剧增加，这时它的吸收作用才比较显著。

（ 二 ） 大气的散射作用

  1. 瑞利散射（分子散射）：短波散射强（天空呈蓝色）。
  2. 米氏散射（气溶胶散射）：粒径与波长可比，致灰白浑浊。
  3. 非选择性散射：粒径远大于波长，各波段近似等强（云雾致亮白）。
